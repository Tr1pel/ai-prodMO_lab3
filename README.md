## Задача 1. Релизовать 3 метода фильтрации данных

1) Реализовать метод скользящего среднего или экспоненциальное сглаживание
2) Реализовать один из подвидов (на свое усмотрение) фильтра Калмона
3) Реализовать один из предложенных фильтров (на свое усмотрение):
    - Фильтр Савицкого-Голея
    - Фильтр Винера
    - Фильтр на основе Спектрального вычитания
    - Фильтры нижних частот и Баттерворт

4) Хард часть: Реализовать предложенный метод:
    - Вейвлет-преобразование
    - Адаптивная фильтрация с использованием алгоритма LMS
